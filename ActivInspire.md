# ActivInspire Install

Download:
[https://sourceforge.net/projects/fabiololix-os-archive/files/Packages/qt5-webengine-5.15.19-4-x86_64.pkg.tar.zst/download]([url](https://sourceforge.net/projects/fabiololix-os-archive/files/Packages/qt5-webengine-5.15.19-4-x86_64.pkg.tar.zst/download))

Download:
[https://sourceforge.net/projects/fabiololix-os-archive/files/Packages/qt5-webchannel-5.15.18%2Bkde%2Br3-1-x86_64.pkg.tar.zst/download]([url](https://sourceforge.net/projects/fabiololix-os-archive/files/Packages/qt5-webchannel-5.15.18%2Bkde%2Br3-1-x86_64.pkg.tar.zst/download))

Run:
```
sudo pacman -U ~/Downloads/qt5-webchannel-5.15.18+kde+r3-1-x86_64.pkg.tar.zst && sudo pacman -U ~/Downloads/qt5-webengine-5.15.19-4-x86_64.pkg.tar.zst && yay -S --needed activinspire && yay -s libxml2
```

Run `inspire` to check for any errors... if errors, fix them on your own, if not close ActivInsipre and open normally
Deleted any downloaded files in Downloads folder that you dont need!
